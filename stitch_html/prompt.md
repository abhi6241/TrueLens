# Google Stitch Prompt --- TrueLens

## Project

Build a **production-quality frontend UI only** for an AI-powered News
Bias & Transparency Platform called **TrueLens**.

### Brand

-   **App Name:** TrueLens
-   **Tagline:** *See Beyond the Headlines*
-   **Logo Concept:** A modern blue camera lens combined with a
    shield/checkmark and subtle globe/news lines, representing
    transparency, trust, AI analysis, and global news. Use a minimal
    flat vector style suitable for a SaaS product.
